TabOrganizer

首次安裝指引

只需做一次。完成後雙擊桌面捷徑即可秒開管理頁。

1
開啟「開發人員模式」
切換到剛打開的 chrome://extensions 分頁，把右上角的「開發人員模式」開關打開。
2
載入未封裝項目
把已開啟的 dist 資料夾（File Explorer 視窗）整個拖曳到 extensions 頁面。
或點頁面左上的「載入未封裝項目」按鈕，選 D:\Desktop\AI\MiniPrograms\TabOrganizer\dist。
3
確認 Extension ID
新出現的 TabOrganizer 卡片應顯示 ID：

eanilmbkohdgpndehpbikchfpnaboloh

如果不一樣，表示拖錯資料夾，請重新拖正確的 dist/。
4
完成！再雙擊桌面 TabOrganizer
關閉本頁與設定對話框，再次雙擊桌面 TabOrganizer 圖示， 這次會在您日常 Chrome 內直接開啟管理頁（無 tab bar、無位址列的 app 視窗）。

之後每次雙擊都是這樣，不再需要任何設定。
WHY ONE-TIME SETUP
Chrome 137+ 出於安全因素停用了 --load-extension 命令列旗標。 對於本地開發的未封裝 extension，唯一可靠的永久安裝方式就是透過 chrome://extensions 的「載入未封裝項目」。 一次設定完成後，extension 在您日常 Chrome 內就會永久保留。

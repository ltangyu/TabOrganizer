TabOrganizer — 一鍵整理瀏覽器分頁，截圖保留、釋放記憶體

長時間使用瀏覽器累積大量分頁是常見痛點：占用記憶體、難以快速回顧曾看過什麼、想關掉又怕丟失。TabOrganizer 透過完整自動流程解決：

✨ 主要功能

• 分頁分類 — 手動建立分類 + 按 domain 自動推薦（支援 *.example.com 萬用字元）
• 截圖歸檔 — 截圖保留 URL 與標題，縮圖存於本機 IndexedDB、原圖下載到資料夾
• 404 自動剔除 — HEAD 檢查失效的不浪費截圖成本，記到剔除清單但分頁仍會關閉
• 一鍵整理 — 所有分頁自動掃描 → 檢查 → 截圖 → 關閉，釋放記憶體
• 縮圖管理頁 — 全螢幕網格瀏覽，可篩選、搜尋、按原分類一鍵重新打開所有頁面
• 定期重掃 — 背景任務週期性檢查已歸檔連結是否仍有效，失效標紅
• 深色模式 — 深灰階介面，狀態燈保留原色
• 10 種語言 — 繁體中文、简体中文、English、日本語、한국어、Español、Français、Deutsch、Português、Tiếng Việt

🔒 完全本機運作

所有資料（分類、截圖、歸檔）都在您的瀏覽器與本機下載資料夾中，**不傳送任何資料到外部伺服器**。詳細隱私政策見 GitHub README。

🎨 視覺設計

遵守 ii Design Language：黑白灰配色（狀態燈是唯一彩色例外）、0.5 px 邊框、10 px 主圓角、Inter + Space Grotesk 字體、無 emoji。

📦 開源

🛠 技術細節

Chrome MV3、Vue 3、TypeScript、IndexedDB、Service Worker。完全本機運作，無第三方追蹤。

⚙️ 設定

可在控制面板自訂截圖延遲、縮圖寬度、定期重掃頻率、卡片網格欄數、密度、縮圖比例等。

💡 適合誰用

• 同時開很多分頁、容易記憶體爆掉的使用者
• 喜歡有條理整理研究資料的學生與研究者
• 希望保留瀏覽紀錄但不想留太多分頁開著的人
• 重視隱私、不希望瀏覽資料被上傳的人
